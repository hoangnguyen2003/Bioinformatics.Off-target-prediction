# Tin sinh học
## Phương pháp Học sâu dự đoán hoạt động ngoài mục tiêu trong chỉnh sửa gen với công nghệ CRISPR/Cas9

Yều cầu:
* Python 3.10.13
* numpy 1.26.4
* tensorflow 2.15.0
* sklearn 1.5.0
* keras 3.2.1
* pandas 2.2.2
* xlrd 1.2.0

Xem trong thư mục ```TrainingProcess``` để xem quá trình huấn luyện cũng như cách chạy chương trình.
